# Release Checklist

Run this checklist before each public release.

## Legal and Privacy

- Privacy Policy matches actual data practices.
- Terms of Service matches subscription and app behavior.
- Privacy rights request path is published.
- App Store privacy label matches the data map.
- Medical disclaimer is visible where health-adjacent copy appears.

## App Store

- Privacy manifest is accurate.
- Subscription products are configured through Apple In-App Purchase.
- Restore purchases works.
- Family Controls and Screen Time entitlements are in place.
- Export compliance answers are reviewed.

## Accessibility

- VoiceOver pass completed.
- Dynamic Type pass completed.
- Reduced Motion pass completed.
- Touch target and contrast pass completed.

## Security

- No secrets in source.
- Vendor register updated.
- Access reviewed for GitHub and Apple Developer.
- Smoke tests pass.

## Evidence

Store test output, screenshots, App Store configuration screenshots, privacy review notes, and release approval notes with the release date.
