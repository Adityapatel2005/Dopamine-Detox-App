# Data Retention

## Current V1 Local Data

Focus sessions, Focus Score inputs, Screen Time app selections, weak-spot prediction inputs, goals, and recap content remain on device in the current scaffold.

## User Control

Users can remove local app data by deleting Lockd from the device. A future in-app reset control should delete local focus history, selected rules, predictions, goals, and recap drafts.

## Subscription Records

Apple manages purchase and subscription records through App Store systems.

## Future Backend Rule

If Lockd adds accounts, sync, support tickets, analytics, or notifications, this document must be expanded with retention periods, deletion workflows, backup handling, and processor responsibilities.
